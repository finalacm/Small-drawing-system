# Introduction
这是我很早之前做的图形学demo。除了实现简单的画笔之外，还加入一些3D物体，并且可以让它们移动、旋转、拉伸和放缩。由于demo是在MFC下实现的，和OpenGL单独实现某个物体渲染的程序是不一样的。如果你需要OpenGL去渲染单个物体的程序，可以联系我。

# IDE 
建议使用Visual Studio IDE（版本> = 2013）
